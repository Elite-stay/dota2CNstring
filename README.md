#dota2字符串变化跟踪项目#

###包含有：###

* dota_schinese.txt
    * 包含游戏中的所有游戏内容文本
    * 比如技能描述、buff描述等
* item_schinese.txt
    * 包含所有物品文本
    * 比如饰品、比赛门票等
* panorama/dota_schinese.txt
	* 包含游戏中的所有界面文本
	* 比如游戏设置、互动指南文本等
* dota_english.txt
* item_english.txt
* panorama/dota_english.txt
    * 这三个文本都是用来对比使用的
    * 通常可以解析大多数游戏改动
    * 全部游戏改动请看[这里](http://git.oschina.net/_stay/GameTracking)
